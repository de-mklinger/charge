# Installation

Install it globally with npm:

```shell
❯ npm install --global @static/charge
```

Or add it to your application’s `package.json`:

```shell
❯ npm install --save-dev @static/charge
```

**Warning!** The `charge` package on npm is _not_ this package. You must use the scoped package name `@static/charge`.
